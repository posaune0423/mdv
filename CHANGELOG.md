## [0.1.0] - 2026-03-29

### Added

- Interactive terminal viewer for GitHub Flavored Markdown (Ghostty / Kitty).
- Plain-text rendering to stdout when stdout is not a TTY.
- Images (PNG, JPEG, GIF, WebP), SVG, syntax-highlighted code blocks, optional Mermaid.
- `--theme`, `--watch`, and `--no-mermaid` CLI flags.
- Prebuilt binaries via GitHub Releases, install script, Homebrew tap, and mise `github:` backend.
